This java program lets you make your own terminal based adventure game!
There are 4 editable files that can be changed to create your own, personal, labrynth.
For the following files: rooms.txt, descriptions.txt, and mobs.txt, every new line is a new room / description / mob
You might want to keep these things generic so it fits in the game better, use the existing files as templates to create your game

For the puzzles.txt, the format should be {Question:key}, with every question and key pair on a new line.
